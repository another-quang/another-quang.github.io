Govtree

GovTree is a web application that allows users to look up Australian Government's departments and personnel. The application features search-as-you-type and auto-complete search functionality. It also displays the organisation chart of the personnel and the departments in an interactive format.

  • My role: Lead designer and developer
  • Technology: Nodejs, Reactjs, Webpack
